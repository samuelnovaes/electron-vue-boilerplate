# Electron and Vue Boilerplate
## Get started
```sh
# Clone the Electron Vue Boilerplate repository
git clone https://github.com/samuelnovaes/electron-vue-boilerplate

# Install dependencies
npm install

# Start the application
npm start

# Build for Windows 32 bits
npm run build win32

# Build for Windows 64 bits
npm run build win64

# Build for Linux 32 bits
npm run build linux32

# Build for Linux 64 bits
npm run build linux64

# Build for MacOS
npm run build mac
```
